## 详细对比

### 1. 上手难度

**方案一（控制台脚本）**：
- 👍 完全不需要安装任何东西
- 👍 复制粘贴就能用
- 👎 每次用都要重复操作

**方案二（油猴脚本）**：
- 👍 只需要装一次扩展，之后自动生效
- 👎 需要先安装Tampermonkey扩展

**方案三（Python脚本）**：
- 👍 有图形界面，操作直观
- 👎 需要安装Python和依赖库
- 👎 首次运行会下载浏览器（约150MB）

### 2. 使用体验

**方案一**：
- 每次都要打开F12控制台
- 页面刷新就要重新运行
- 状态显示比较简单

**方案二**：
- 访问页面自动加载
- 右下角有浮动按钮随时可用
- 有漂亮的控制面板和进度条
- 配置自动保存
- 快捷键 Ctrl+E 快速打开

**方案三**：
- 独立的GUI程序
- 实时日志显示
- 可以保存账号密码，一键全自动
- 支持后台无头运行
- 可以随时停止

### 3. 功能完整性

| 功能 | 控制台 | 油猴 | Python |
|------|--------|------|--------|
| 自动识别未评估项 | ✅ | ✅ | ✅ |
| 自动填写单选 | ✅ | ✅ | ✅ |
| 自动填写主观评价 | ✅ | ✅ | ✅ |
| 绕过2分钟等待 | ✅ | ✅ | ✅ |
| 自动提交 | ✅ | ✅ | ✅ |

## 推荐方案

### 🏆 推荐大多数用户使用：方案二（油猴脚本）

**理由：**
1. **平衡了易用性和功能** - 一次安装，后续使用非常方便
2. **体验最好** - 漂亮的界面，流畅的操作
3. **不需要折腾Python环境** - 普通用户也能轻松上手
4. **功能足够用** - 包含了所有需要的自动化功能
5. **安全可靠** - 代码透明，在浏览器沙箱中运行

**适合人群：** 不想折腾环境，希望打开浏览器点一下就能用的同学

---

### 🚀 推荐临时使用/尝鲜：方案一（控制台脚本）

**理由：**
1. 最快上手，10秒就能开始用
2. 不需要安装任何东西
3. 适合偶尔用一次的场景

**适合人群：** 就想用一次，不想装任何扩展/软件的同学

---

### 💻 推荐技术党/追求极致：方案三（Python脚本）

**理由：**
1. 功能最强大，最稳定
2. 可以完全自动化（包括自动登录）
3. 支持后台运行，可以一边跑一边干别的
4. 可以扩展更多功能（定时、多账号等）

**适合人群：** 
- 熟悉Python的同学
- 需要经常评教（比如帮同学也弄）
- 希望完全自动化、不想碰浏览器的
- 想在此基础上二次开发的

## 快速开始指南

### 如果你选方案一（控制台脚本）
1. 打开 [方案一文档](./方案一-浏览器控制台脚本.md)
2. 登录教务系统，进入评估列表页
3. F12打开控制台，粘贴脚本按回车
4. 等待完成即可

### 如果你选方案二（油猴脚本）
1. 安装 Tampermonkey 扩展
2. 打开 [方案二文档](./方案二-油猴脚本Tampermonkey.md)
3. 复制脚本代码，在油猴中新建脚本粘贴保存
4. 访问教务系统评估页面，点击右下角机器人按钮
5. 选择策略，点击开始

### 如果你选方案三（Python脚本）
1. 安装 Python 3.8+
2. 执行 `pip install playwright && playwright install chromium`
3. 打开 [方案三文档](./方案三-Python自动化脚本.md)
4. 将代码保存为 `auto_eval.py`
5. 运行 `python auto_eval.py`，填写账号，点击开始

## 重要提示

1. **评分安全**
   - 建议使用"随机好评"策略，看起来更真实
   - 不建议全部选最高分/最低分，太假
   - 主观评价已经内置了8条不同的好评，随机选择

2. **使用风险**
   - 脚本仅供学习交流使用
   - 请认真对待教学评估，这是老师改进教学的重要参考
   - 使用本脚本产生的任何后果由使用者自行承担

3. **脚本失效怎么办**
   - 如果学校更新了页面结构，脚本可能失效
   - 到时候按F12看看控制台有什么错误，可以尝试自己修改选择器
   - 或者检查是否有更新版本
