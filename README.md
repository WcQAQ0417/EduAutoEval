# 🤖 教务系统自动评教助手

自动填写教学评估问卷，等待2分钟倒计时后自动提交，完成一门后返回列表继续处理下一门，直到全部完成。

## ⚠️ 重要说明

- **后端有服务端时间校验**（记录进入页面时间），无法绕过2分钟等待，脚本会真实等待倒计时结束
- 等待期间每10秒打印一次剩余时间，请耐心等待
- 脚本默认填写好评（随机A/B/C，不会全选A/E，更真实）
- 主观评价使用预设的无空格好评（教务系统不允许空格）

## 🚀 快速开始

### 方案一：浏览器控制台脚本（零安装）

1. 登录教务系统，进入「教学评估」列表页
2. 按 `F12` 打开开发者工具 → 切换到 Console（控制台）标签
3. 打开 [console-script.js](./js-scripts/console-script.js)，复制全部内容
4. 粘贴到控制台，按回车运行
5. 右上角会出现状态面板，自动开始处理

详细说明 → [js-scripts/README.md](./js-scripts/README.md)

### 方案二：油猴脚本（推荐）

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展（Chrome/Edge/Firefox都支持）
2. 点击 Tampermonkey 图标 → 添加新脚本
3. 删除默认内容，粘贴 [tampermonkey-script.user.js](./js-scripts/tampermonkey-script.user.js) 的全部内容
4. 按 `Ctrl+S` 保存
5. 访问教务系统评估页面，右下角会出现 🤖 浮动按钮，点击打开面板后点「开始」

详细说明 → [js-scripts/README.md](./js-scripts/README.md)

### 方案三：Python 脚本（最适合分享）

**Mac/Linux：**
```bash
cd python-implementation
bash start.sh
```

**Windows：**
双击 `start.bat` 即可

脚本会自动安装 uv、同步依赖（清华源加速）、安装 Chromium 浏览器，然后启动程序。启动后在打开的浏览器中手动登录即可。

详细说明 → [python-implementation/README.md](./python-implementation/README.md)

## 🔧 核心工作流程

```
┌─────────────────────────────────────────────────────────────┐
│                        评估列表页                            │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐                     │
│  │ 未评估1 │  │ 未评估2 │  │ 已评估  │  →  找到"评估"按钮    │
│  └────┬────┘  └─────────┘  └─────────┘                     │
└───────┼─────────────────────────────────────────────────────┘
        │ 点击进入
        ▼
┌─────────────────────────────────────────────────────────────┐
│                        问卷填写页                            │
│  1. 自动选择单选框（JS设置checked+触发事件）                  │
│  2. 自动填写主观评价（无空格好评，随机选一条）                │
│  3. 等待2分钟倒计时（每秒检查RemainM/RemainS/flag）           │
│  4. 倒计时结束（flag=true 且 0分0秒）→ 用jQuery AJAX提交     │
│  5. 自动更新token，处理返回结果                              │
└───────┬─────────────────────────────────────────────────────┘
        │ 成功后导航回列表页
        └───────────────────────────────────────────┐
                                                    ▼
                                          回到列表页，找下一个
                                          直到没有待评估项目
```

## 🐛 已知问题与排障

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 点击单选框超时 | Ace Admin样式隐藏了原生input | 已修复：使用JS直接设置checked，不依赖可见性 |
| "保存失败,非法提交" | token过期/时间未到 | 已修复：用原生AJAX提交，自动更新token；等待真实2分钟 |
| 提交成功后退出不继续 | 没有返回列表页 | 已修复：提交后主动导航回列表页，继续循环 |
| 2分钟绕过失败 | 后端有服务端时间校验 | 不再尝试绕过，真实等待2分钟，100%可靠 |
